# PingPON

A dark themed interactive site that shows an AI learning to play ping pong using a genetic algorithm. Each visitor gets their own session and can watch the neural network improve over generations.

## Features

- Real time canvas game
- Evolving neural network controller
- Live neural link visualization
- Session saved between visits using localStorage
- Learn page explaining genetic algorithms
